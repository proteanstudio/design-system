Checkbox

Checkboxes and toggles allow the user to select one or more options from a group, or to switch between two states.

Best practices

Checkboxes vs. radio buttons

  • Use checkboxes when the user can select more than one option in a group.
  • Use radio buttons when the user can select only one option in a group.
    • Note: the design system has not implemented a radio button element yet. Native radio buttons with a fieldset will work, but they do not have design system styles associated with them at this time.

Checkboxes vs. toggle switches

  • Use a checkbox when the selection must be confirmed before taking effect (typically by clicking a form button).
    • For example, a checkbox is appropriate when inviting a user to opt-in to email communications during a sign up workflow. The email opt-in, along with other sign up information, is collected only after the user clicks a button (e.g. "Complete registration").
  • Use a toggle switch when the selection should take effect immediately, without a separate confirming action.
    • For example, applications that offer a "dark mode" could use a toggle switch to enable and disable that mode. The toggled setting takes effect immediately.

API

Properties

In the following examples, element refers to an instance of protean-checkbox (i.e. element = document.querySelector('protean-checkbox#myCheckbox')).

  • variant

    "checkbox" (default) | "toggle"
    Attribute name: variant
    The visual style of the checkbox.
    element.variant = 'toggle';

  • checked

    boolean
    Attribute name: checked
    The checked state of the checkbox.
    element.checked = true;

  • label

    string
    Attribute name: label
    The visible label for the checkbox.
    element.label = 'Field label';

  • a11yLabel

    string
    Attribute name: a11y-label
    The label read by assistive technologies for the checkbox; a11yLabel is used here instead of ariaLabel to prevent double readings of the text by those technologies. Note: when label is set, this value will not be bound to the field.
    element.a11yLabel = 'Field aria-label';

  • disabled

    boolean
    Attribute name: disabled
    Disables checkbox interaction.
    element.disabled = true;

  • alignment

    "left" (default) | "right"
    Attribute name: alignment
    The placement of the checkbox relative to its label.
    element.alignment = "right";

  • indeterminate

    boolean
    Attribute name: indeterminate
    Shows the checkbox in an indeterminate state. Best used to indicate partial selection of options on a checkbox that controls all others in a group. Does not apply to toggles.
    element.indeterminate = true;

Event

  • change

    Fires when the checkbox is toggled.
    Event detail
    {{` event: { detail: { checked: boolean, } } `}}
    {{` function onChange(event) { this.checked = event.detail.checked; } `}} {{` `}}

CSS variables

Listed below are the CSS variables that can be adjusted to alter the protean-checkbox element's appearance along with the default values set in the protean-styles package:

  • --protean-checkbox-margin

    The margin around the protean-checkbox element.
    --protean-checkbox-margin: 1rem;

  • --protean-checkbox-outline-color

    The color of the outline around the checkbox.
    --protean-checkbox-outline-color: var(--action-color);

  • --protean-checkbox-toggle-path-color

    The color of the toggle path when not checked.
    --protean-checkbox-toggle-path-color: #535353;

  • --protean-checkbox-toggle-circle-fill

    The color of the toggle circle when not checked.
    --protean-checkbox-toggle-circle-fill: #000000;

  • --protean-checkbox-checked-outer-color

    The color that fills a standard checkbox and the toggle path when checked.
    --protean-checkbox-checked-outer-color: var(--action-color);

  • --protean-checkbox-checked-inner-color

    The color of a standard checkbox's check mark and the toggle circle when checked.
    --protean-checkbox-checked-inner-color: var(--action-contrast-color);

  • --protean-checkbox-disabled-opacity

    The opacity of the field while it is disabled.
    --protean-checkbox-disabled-opacity: var(--disabled-opacity);
