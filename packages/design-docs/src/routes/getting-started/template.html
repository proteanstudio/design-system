Getting started

Follow the steps below to install the design system elements and styles. If you notice any problems or if you would like to submit a request, please open an issue in the Github repository. Please note that all system features and element properties are subject to change until a 1.0.0 version is released. Changes will be listed the Release Notes (coming soon).

We are currently working through some issues with this path. Once resolved, we will add the accompanying documentation. Please use the static imports until then.

Use on a static site

Static versions of the elements and styles are available for download from unpkg using link and style tags.

Notes

  • Protean elements are web components written using Typescript and StencilJS.
    • All element types are declared in the global namespace - the benefits of this surface when querying an element via document.querySelector and when using an element in a .tsx file.
  • React has some deficiencies that need to be accounted for.
    • The StencilJS integration docs provide some greater detail and guidance on this matter.
